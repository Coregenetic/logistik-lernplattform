# 🚛 Lernplattform Logistik

> **Die digitale Lernplattform für die Klasse LOG25a** – erstellt von [Coregenetic](https://github.com/coregenetic)

---

## 📖 Über das Projekt

Die **Lernplattform Logistik** ist eine webbasierte Lernapp, die speziell für Auszubildende der Klasse **LOG25a** entwickelt wurde. Sie hilft dabei, sich gezielt auf Klassenarbeiten, Tests und Prüfungen vorzubereiten – direkt im Browser, ohne Installation.

---

## ✨ Features

- 📚 **Lernfelder** – strukturierte Inhalte nach Lernfeldern (LF 1–12)
- 📘 **Fächer** – Englisch, Kommunikation & WiPo mit Kapiteln und Inhalten
- ❓ **Quiz-System** – KI-gestütztes Bewertungssystem mit Keyword-Erkennung
- 🗂 **Vokabeltrainer** – Englisch ↔ Deutsch mit Fortschrittsverfolgung
- ✅ **Fortschritt** – persönlicher Lernfortschritt wird automatisch gespeichert
- 📊 **Dashboard** – Überblick über erledigte Inhalte und Lernfelder
- 📱 **Responsiv** – vollständig für Desktop und Mobile optimiert

---

## 👥 Rollen

| Rolle | Beschreibung |
|---|---|
| `azubi` | Standardrolle für alle Azubis der LOG25a |
| `mod` | Kann Inhalte, Kapitel und Quizze verwalten |
| `admin` | Vollzugriff inkl. Benutzerverwaltung und Freischaltungen |
